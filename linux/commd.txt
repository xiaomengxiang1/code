-----------------------linux命令---------------

[]里面的表示选填
commd [option] [parameter]
命令    可选参选项    参数

ls linux路径
ls 查看文件夹(目录)的内容,没参数,默认为当前工作目录的内容
-a  展示出隐藏文件  all
-l  列表展示内容,有更多的细节   list
-h  和-l搭配使用,显示文件夹大小


cd change directory
cd [路径]
无参数表示回到HOME目录

print work directory查看当前所在文件夹,无选项，无参数
pwd


相对路径    /开头
绝对路径    当前工作目录下开始

特殊路径符
.   表示当前目录    cd ./Desktop    表示切换到当前目录下的Desktop目录内 和cd Desktop效果阴一样
..  上一级目录      cd ..   切换到上一级目录    cd ../..    上一级的上一级
~   HOME目录
~/Desktop   等同于  /home/ljy/Desktop

进入home目录会自动进入用户名目录里，这是系统设定的记住就行，
目的是为了保证当多个用户使用同一台机器时每个用户的home目录是唯一的，
并且可以通过用户名来访问

make dictionary 创建新目录

mkdir [-p] linux路径
mkdir -p 一次性创建多层次目录

路径必填,相对或绝对

创建新的文件
mkdir是创建新的目录，touch是创建新的文件  touch 命令无选项，参数必填，
touch linux路径

cat 用于查看文件内的内容
cat linux路径

cat把全部内容显示出来，more支持翻页
more linux路径

空格翻页
按q退出

复制
cp [-r] 参数1 参数2
-r  复制文件夹(目录)使用,表示递归
参数1   linux路径   表示被复制的文件或文件夹
参数2   linux路径   要复制去的地址

移动
mv 参数1 参数2
参数1   linux路径   表示被移动的文件或文件夹
参数2   linux路径   要移动去的地址

删除
rm [-r -f] 参数1 参数2 ... 参数n
-r  删除文件夹
-f  强制删除
参数1 参数2 ... 参数n   要删除的文件或文件夹的路径

*是通配符,即匹配任意内容
test*,表示匹配任意以test开头的内容
*test,表示匹配任意以test结尾的内容
*test*,表示匹配任何包含test的内容

su - root    切换至root用户
exit        退出root用户

查看命令程序的文件存放位置
which   要查找的命令

find    -按文件名查找文件
find 起始路径 -name "被查找文件名"

find    -按文大小查找文件
find 起始路径 -size +|-n[KMG]
+ - 表示大于和小于
n表示数字大小
KMG表示大小单位,k(小写字母)表示kb,M表示MB,G表示GB

例:
查找小于10kb的文件  find / -size -10k
查找大于100mb的文件  find / -size +100M
查找大于1gb的文件  find / -size +1G


grep命令    从文件中通过关键字过滤文件行
grep [-n] 关键字 文件路径
-n 可选参数 表示在结果中显示匹配的行的行号
关键字  表示要过滤的字,有空格或者特殊符号建议用" "将关键字包围起来
文件路径    要过滤的内容的文件路径,可作为图形输入端口
